About

The InterCrypto smart contract provides a decentralized service on the Ethereum blockchain to convert Ether to other cryptocurrencies. The InterCrypto smart contract uses the Oraclize service to create a request to ShapeShift to execute this conversion. This website has a simple interface to use InterCrypto (above) as well as a demo Wallet smart contract (below) which shows an example of how InterCrypto is intended to be used.

The contract has been rigorously tested and examined throughout its development as part of an academic project at Imperial College London by Jack Tanner investigating mechanisms for on-blockchain cryptocurrency conversion. The academic report, including a thorough survey of smart contract capable blockchains, cryptocurrency conversion technology and the development report for InterCrypto can be downloaded here.

The InterCrypto smart contract code was casually reviewed by smart contract expert Laurence Kink who found no major problems with the code. The InterCrypto author, Jack Tanner, considers the contract ready for use and users should read the academic report for more details.

The InterCrypto smart contract exists on the Ethereum blockchain 'as is' without warranty of any kind, either express or implied by the author or any other party, including, but not limited to, the implied warranties of fitness for a purpose, or the warranty of non-infringement.

This website is open-source and runs client-side only. No keys are ever sent to or stored by intercrypto.org. You can check out the website source code for this page on Github and run it locally if you like.

Using InterCrypto

InterCrypto is deployed to the Ethereum mainnet and Rinkeby and Ropsten testnets which can be connected to and used by importing the InterCrypto Interface solidity file and then inheriting the usingInterCrypto contract as shown below:

import "github.com/ugmo04/inter-crypto/contracts/InterCrypto_Interface.sol";

contract MySmartContract is usingInterCrypto {
     function myConvert(_coinSymbol, _toAddress) payable external {
          uint conversionID = intercrypto_convert(msg.value, _coinSymbol, _toAddress);
     }
}
