About

The InterCrypto DAPP provides a decentralized service on the Ethereum blockchain to convert Ether to other crypto-currencies. The InterCrypto DAPP uses the Oracalize DAPP to create a request to ShapeShift to execute this conversion. This page shows a demo Wallet DAPP that exhibits the use case of InterCrypto and also gives direct access to call functions on InterCrypto itself. Connecting to InterCrypto require smart contract coders to import the InterCrypto Interface by including the following line in their solidity smart contract.

import "github.com/ugmo04/inter-crypto/contracts/InterCrypto_Interface.sol";
See an example here

This website is open-source and runs client-side only. No keys are ever sent to or stored by intercrypto.org. You can check out the website source code for this page on Github and run it locally if you like.
